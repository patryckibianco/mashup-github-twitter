## mashup-github-twitter

###### This is a sample project that consumes the search service from GitHub and Twitter APIs to make a mashup of both platforms

**Usage**

    1 - Place the application.properties file at the same folder where JAR file was generated
    2 - Edit the application.properties file with your own Twitter bearer token and set a query param filter 
    3 - Run the following command line to execute the mashup service (it needs to be executed with Java 8 JDK)
        
        java -jar name-of-generated-jar.jar
